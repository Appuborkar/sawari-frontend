# Sawari – Online Bus Booking Platform

###  Live Demo  
 [https://sawari-bus-booking.vercel.app/](https://sawari-bus-booking.vercel.app/)

---

## Project Overview
**Sawari** is a full-stack **MERN (MongoDB, Express.js, React.js, Node.js)** based **online bus booking system** that allows users to search, select, and book bus tickets easily.  
The platform provides a seamless user experience with real-time seat selection, temporary seat hold, ticket booking, and cancellation functionalities.

---

##  Features

### User Features
- **Search Buses:** Search buses by source, destination, and date.  
- **Seat Selection:** Interactive seat layout for choosing seats in real time.  
- **Booking System:** Temporary seat hold → confirm booking → ticket generation.  
- **Ticket Cancellation:** Cancel ticket if more than 4 hours remain before departure.  
- **Profile Management:** View and update user profile details.  
- **View Tickets:** Access all booked and cancelled tickets anytime.  

---

## Tech Stack

| Layer | Technologies Used |
|--------|------------------|
| **Frontend** | React.js, React Router, Context API, Axios, Custom CSS |
| **Backend** | Node.js, Express.js |
| **Database** | MongoDB (Mongoose ORM) |
| **Deployment** | Frontend – Vercel, Backend – Render |
| **Others** | Cloudinary (for image upload), JWT (for authentication) |

---

## Core Functionalities
- **Authentication:** Secure login/signup using JWT tokens.  
- **Database Management:** Stores bus, user, and ticket data in MongoDB.  
- **Seat Hold Logic:** Temporarily holds seats before confirmation to prevent double booking.  
- **API Integration:** Clean RESTful APIs for frontend–backend communication.  
- **Responsive UI:** Optimized for desktop and mobile devices.
